* In order to begin, specify the URL in Oracle with the define command eg.
        DEFINE cl43=S:\CS342\project
        @&cl43\create
* The create will then create an account and log onto it for you. From then on, the schema, and database is loaded onto OracleXE by auto activating @data, and @schema.
* Procedures and triggers can be added from @procedures, and @triggers command.
* Sample queries can be run as well with the @queries command.
* If you wish to erase the database, run the @drop command
